.container-table provides a better option to to vertical-align: middle for col-*

Note: You need to add .vertical-middle text-center and define height to make in align it in middle.
EXAMPLE

    <style>
        @media (min-width: 768px) {
            #contable {
                height: 200px;
            }
        }
        #contable,
        #contable .row,
        #contable .row > div {
            border: 1px solid #333;
        }
    </style>

    <div class="container-table" id="contable">
        <div class="row vertical-middle text-center" >
            <div class="col-sm-4 col-md-8 col-lg-4 text-center">
                <p>Some content</p>
                <p>Some content</p>
                <p>Some content</p>
            </div>
            <div class="col-sm-4 col-md-4 col-lg-4 text-center">
                <p>Some content</p>
                <p>Some content</p>
            </div>
            <div class="col-sm-4 hidden-md col-lg-4 text-center">
                <p>Some content</p>
                <p>Some content</p>
                <p>Some content</p>
                <p>Some content</p>
            </div>
        </div>
    </div>
Note: .container-table can only manage 12 columns in a .row. further rows are required to adjust additional columns
