Buttons

Outline

Button Outline is a long awaited feature of bootstrap now added in bootstrap v4. As v4 is in alpha stage, here is a quick support by this project.

EXAMPLE

  <button class="btn btn-default btn-outline" >Button</button>
  <button class="btn btn-primary btn-outline" >Button</button>
  <button class="btn btn-success btn-outline" >Button</button>
  <button class="btn btn-info    btn-outline" >Button</button>
  <button class="btn btn-warning btn-outline" >Button</button>
  <button class="btn btn-danger btn-outline" >Button</button>

Responsive Block

It is favorable to have a full-width button for mobile devices. Hence, .btn-block-xs allows you to configure full-width button just for mobile devices.

Note: .btn-block-xs will only work on xs devices (i.e. mobile) and the class will not be used in other breakpoint. 
<button class="btn btn-primary btn-block-xs" >Button</button>

Rounded

.btn-rounded class converts left and right edges to rounded shape

EXAMPLE

  <button class="btn btn-default btn-rounded" >Button</button>
  <button class="btn btn-primary btn-rounded" >Button</button>
  <button class="btn btn-success btn-rounded" >Button</button>
  <button class="btn btn-info    btn-rounded" >Button</button>
  <button class="btn btn-warning btn-rounded" >Button</button>
  <button class="btn btn-danger btn-rounded" >Button</button>

Circle

.btn-circle enable you to create buttons with icons. 

EXAMPLE





  <button class="btn btn-default btn-circle btn-lg" ><i class="glyphicon glyphicon-plus" aria-hidden="true"></i></button>
  <button class="btn btn-primary btn-circle btn-lg" ><i class="glyphicon glyphicon-music" aria-hidden="true"></i></button>
  <button class="btn btn-success btn-circle btn-lg" ><i class="glyphicon glyphicon-chevron-up" aria-hidden="true"></i></button>
  <button class="btn btn-info btn-circle btn-lg" ><i class="glyphicon glyphicon-chevron-down" aria-hidden="true"></i></button>
  <button class="btn btn-warning btn-circle btn-lg" ><i class="glyphicon glyphicon-volume-off" aria-hidden="true"></i></button>
  <button class="btn btn-danger btn-circle btn-lg" ><i class="glyphicon glyphicon-volume-up" aria-hidden="true"></i></button>

Action

Action button is a floating button at bottom right or left corner. The concept of floating Action button is from Android material design. These action button is also helpful in creating a "back to top" similar to one on this page

  
    <button class="btn btn-default btn-circle btn-lg action-btn" >
        <span class="glyphicon glyphicon-plus" aria-hidden="true"></span>
    </button>
