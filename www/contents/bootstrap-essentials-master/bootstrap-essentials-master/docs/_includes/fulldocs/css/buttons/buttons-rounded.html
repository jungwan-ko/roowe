Rounded

.btn-rounded class converts left and right edges to rounded shape

EXAMPLE

  <button class="btn btn-default btn-rounded" >Button</button>
  <button class="btn btn-primary btn-rounded" >Button</button>
  <button class="btn btn-success btn-rounded" >Button</button>
  <button class="btn btn-info    btn-rounded" >Button</button>
  <button class="btn btn-warning btn-rounded" >Button</button>
  <button class="btn btn-danger btn-rounded" >Button</button>
