Circle

.btn-circle enable you to create buttons with icons. 

EXAMPLE





  <button class="btn btn-default btn-circle btn-lg" ><i class="glyphicon glyphicon-plus" aria-hidden="true"></i></button>
  <button class="btn btn-primary btn-circle btn-lg" ><i class="glyphicon glyphicon-music" aria-hidden="true"></i></button>
  <button class="btn btn-success btn-circle btn-lg" ><i class="glyphicon glyphicon-chevron-up" aria-hidden="true"></i></button>
  <button class="btn btn-info btn-circle btn-lg" ><i class="glyphicon glyphicon-chevron-down" aria-hidden="true"></i></button>
  <button class="btn btn-warning btn-circle btn-lg" ><i class="glyphicon glyphicon-volume-off" aria-hidden="true"></i></button>
  <button class="btn btn-danger btn-circle btn-lg" ><i class="glyphicon glyphicon-volume-up" aria-hidden="true"></i></button>
